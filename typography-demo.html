Text Styles

Lead Text - Used for introductory paragraphs and important content that needs emphasis.

18px / 1.125rem - Relaxed line height

Body Text - The standard text size for most content. This is the default paragraph text that provides good readability and comfortable reading experience.

16px / 1rem - Relaxed line height

Small Text - Used for secondary information, captions, and less important details.

14px / 0.875rem - Normal line height

Caption Text - For fine print, metadata, and minimal text elements.

12px / 0.75rem - Normal line height
